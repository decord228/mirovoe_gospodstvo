# Мировое Господство - Веб-интерфейс

Веб-интерфейс для управления игрой "Мировое Господство"

## Как запустить локально

1. Запустите `start_server.bat`
2. Откройте http://localhost:8000

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы
3. Settings → Pages → Source: main branch
4. Сайт будет доступен по адресу: https://username.github.io/repo-name

## Файлы

- `index.html` - главная страница
- `app.js` - логика приложения
- `sync.js` - синхронизация данных
- `styles.css` - стили
- `data.json` - база данных игры
